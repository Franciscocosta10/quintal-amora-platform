# QuintalAmora-platform
Plataforma web para gestão e engajamento de visitantes do evento Quintal da Amora

O evento Quintal da Amora enfrenta algumas dificuldades na organização, comunicação e controle de atividades em tempo real.

Meu objetivo com esse projeto é centralizar a gestão do evento, melhorar a comunicação e otimizar a experiência dos participantes.

Funcionalidades:
- Check-in por geolocalização
- Programação do evento
- Notificações em tempo real
- Pedidos para retirada
- Catálogo de lojas e artistas
